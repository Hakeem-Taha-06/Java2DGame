# 2D Adventure Game
an old game project of mine made in mid 2025

### Controls:
- WASD to move
- E to interact with objects and pick them up or open doors (u can even heal when you're next to the lake : D)
- F to attack with your sword (aka summon a png of a sword in the direction you are facing in that collides with enemies...)

### Goal:
get to the chest in the top left by using keys, use the keys in the wrong order and YOU WILL GET SOFT LOCKED HAHAHA!!

there are also some slimes you can kill in the beginning (they can't kill u tho and if they knock u into a wall u get stuck heh)
